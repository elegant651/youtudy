# Youtudy

## Description
```
It is a service that allows you to create a learning curriculum with YouTube videos, and share them with others.
```

## Tech stack
```
VueJS, VueX, Vue-CLI, Firebase, Firebase-Auth, Youtube API
```

## Project setup
```
npm install
```
